瞳学 AI · 技术工作流

端侧轻量模型 + 云端大模型 → 实时检测 + 智能分析

1 📸 端侧实时检测 (2fps)
在精灵本体运行，全部本地推理，不上传视频流
  • MediaPipe Face Mesh 人脸检测 + 视线方向 (30fps)
  • MoveNet Pose 身体姿势估计 (20fps) → 趴/歪/正
  • SSD-MobileNet 手机/平板物体检测 (15fps)
  • 距离算法 面部框大小 → 估算眼睛距桌面距离
⬇ 每5秒输出状态向量 ⬇
2 ☁️ 云端 AI 分析
收到状态序列 → 综合分析 → 决策
  • GPT-4o-mini 专注度判定 + 状态分类 (专注/走神/离开)
  • 自定义规则引擎 灰度判断逻辑 (见Prompt设计)
  • 时序分析 5分钟窗口滑动判定疲劳/模式
⬇ 决策结果 ⬇
3 🎯 多端输出
决策结果分发到不同终端
  • 🦉 精灵表情 + 语音 实时提醒孩子（不打断专注）
  • 📱 家长端推送 走神通知 + 每日报告 + 周报
  • 📊 数据仪表盘 学习趋势 + 坐姿分析 + 专注模式

🧠 核心 Prompt 设计（云端 AI）

你是瞳学AI的学习监督助手，名叫小智。 输入：过去5分钟的学生行为状态序列，格式如下： [{t: "14:01:00", face: true, gaze: "book", posture: "upright", distance: 35, phone: false}, {t: "14:01:05", face: true, gaze: "book", posture: "upright", distance: 35, phone: false}, ...] 请判断： 1. 总体状态（专注/轻微走神/明显分心/离开座位） 2. 走神原因（玩手机/发呆/离开/东张西望） 3. 是否需要提醒？若需要，选择提醒方式 4. 距离是否过近 (<30cm)？姿势是否正确？ 5. 5分钟内专注时长占比 输出JSON： {"state": "focused|distracted|away|phone", "confidence": 0-100, "alert_needed": true|false, "alert_type": "voice|parent|silent", "distance_warn": true|false, "posture_warn": true|false, "focus_minutes": 0-5, "summary": "一句话总结"}

📊 严重程度判定规则（端侧+云端融合）

等级条件动作
🟢 正常 连续 ≥ 3分钟 face=true, gaze=book, posture=upright 沉默 / 每15分钟轻声鼓励
🟡 轻微走神 gaze!=book 持续 30-90秒 精灵打哈欠动画 或 轻声提醒"我快睡着了~"
🟠 明显分心 phone=true 或 gaze!=book >90秒 精灵转身不理人 + push家长通知
🔴 离开座位 face=false >120秒 push紧急通知 + 记录离开时长
🔴 距离过近 distance < 30cm 持续 >30秒 精灵担忧表情 + 语音"太近了，对眼睛不好哦"
🟡 姿势异常 posture!=upright 持续 >60秒 精灵歪头模仿 + "你现在的姿势和我好像~"
